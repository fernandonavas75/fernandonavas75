## 📌 Proyectos destacados

### 🧾 PymeFlowEc

Sistema web tipo **ERP/SaaS multitenant** orientado a pequeñas y medianas empresas del Ecuador.

**Características principales:**

* Gestión de empresas
* Gestión de usuarios y roles
* Módulos activables por empresa
* Clientes, proveedores y productos
* Inventario
* Ventas y facturación
* Auditoría de acciones
* Validación de cédula y RUC ecuatoriano
* Recuperación de contraseña por correo
* Despliegue en AWS

**Tecnologías usadas:**

```txt
Frontend: Angular / React
Backend: Node.js + Express
Base de datos: PostgreSQL
Autenticación: JWT
Cloud: AWS
```
